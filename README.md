# AI-Based Resume Screening System using Machine Learning and Excel

## Project Overview
The AI-Based Resume Screening System is a machine learning project that automates the resume evaluation process. It predicts whether a candidate is likely to be selected based on features such as skills, experience, education, certifications, salary expectation, project count, and AI score.

The project also includes an interactive Excel dashboard for visualizing candidate statistics and recruitment insights.

---

## Features

- Resume selection prediction using Machine Learning
- Data preprocessing and feature engineering
- Model training and evaluation
- Excel Dashboard with KPIs and charts
- Streamlit Web Application for prediction
